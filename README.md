Ventee mobile application

### Deploy

```shell script
git clone https://github.com/ventee-app/ventee-mobile
cd ./ventee-mobile
nvm use 16
npm i
```

### Launch

```shell script
npm start
```

Emulators:

```shell script
npm run android
```

```shell script
npm run ios
```
